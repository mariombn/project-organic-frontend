# project-organic-frontend

Projeto para organização pessoal baseado no notion do Reservatório de Dopamina

## Project setup
```
npm install
cp .env.exemple .env
npm run serve
```

## Compiles

#### Compiles and hot-reloads for development
```
npm run serve
```

##### Compiles and minifies for production
```
npm run build
```

##### Lints and fixes files
```
npm run lint
```
